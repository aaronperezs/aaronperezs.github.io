Hikin: Madrid Mobile App Case Study

Improving my first Android app!

Length : 1-2 weeks

Role: UX designer designing from conception to delivery

Responsibilities: Conducting interviews, paper and digital wireframing, low and high-fidelity prototyping, conducting usability studies, accounting for accessibility, and iterating on designs.

Aaron Perez

Project Summary

This is the redesign of the first app I developed. Back then, I used Android Studio and its default components to create the app, now we will apply some UX principles to the design process. In this project, we are going to see how we defined the problems, the procedures we used and the results.

The app is a finder of hiking trails from the Autonomous Community of Madrid. All the information about the trails is available in a JSON file format in datos.comunidad.madrid.

The Challenge

The app

With this new redesign, I wanted to create a more simple and usable experience for the user. Simplicity and usability are the two main objectives of the project. As it was originally created with the default android components, I also want to create a new identity for the app. Finally, I think some new features can be added to the app so we ensure the app quality.

  • Create a new brand identity and maintain visual consistency.
  • Add some new features to the app so the overall experience will be enhanced.
